How to Use

  • Step 1:

      Upload a file in the top left input section. The file will be read and all the columns will be displayed as draggable buttons.

  • Step 2:

      Drag and drop columns in the section marked by a dashed line. The data should appear in the table below.

  • Step 3:

      You can select any row/column/cell in the displayed table, the chart selector in the top right will update as your selected data changes.

  • Step 4:

      The primary column for your chart can be toggled by clicking the small button on the table column header. If there is no primary column, it is taken as table ID by default.

  • Step 5:

      Click on an enabled chart button to view your visualization!
